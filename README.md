# SMIT GPA Calculator

A sleek, dark-themed GPA calculator built specifically for the **Sikkim Manipal Institute of Technology (2025-26)** curriculum. 
    
This tool allows students to select their engineering branch and semester, automatically loading the correct subjects and credit values to calculate their Semester GPA accurately.

## 🚀 Features

- **Curriculum Accurate:** Contains updated subject data and credit points for major branches (CSE, ECE, Mechanical, Civil, IT, etc.).
- **First-Year Support:** Includes specific calculations for both Physics and Chemistry study cycles.
- **Dynamic Calculation:** Automatically recalculates your GPA as you select and update your grades.
- **Modern UI:** Responsive, dark-mode design built with Tailwind CSS.

## 🛠️ Tech Stack 

- **HTML5**
- **Tailwind CSS** (via CDN for styling)
- **Vanilla JavaScript** (Logic & DOM Manipulation)
